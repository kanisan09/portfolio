Java portfolio

GitHub

動的アプリケーション「のんねむ」

https://github.com/kanisan09/Java-portfolio.git

説明

サーブレットとjspを使用してつぶやき機能がある動的アプリケーションを作りました。

使用したもの

統合開発環境 Eclipse / tomcat

使用サーバー

AWS EC2

インスタンスやポートの開放設定、Amazon Linux 2023にmysqlやtomcatをインストール、起動確認などを経て無事サーバーに上げることはできましたが毎日不安との戦いでした。
無駄な事もたくさんしましたしかなり時間をかけてしまいましたがこの経験は一生忘れないと思います。

制作時間

1ヵ月半

反省点、改善点

動的アプリケーションがどのように作られているのか理解はできたが、セッションがうまくきいておらず投稿文がスコープの中に残ったままになってしまった。
H2データベースを練習でつないでみたり、結果やりたい事を詰め込みすぎて結局初めから作り直す事態にもなりながら、次挑戦する際には計画をしっかり練った上でMYSQLなどのDBをつないで、ログインや投稿内容がうまく記憶できるようにしたいです。
DAOやxmlファイルの理解も深めたい。フレームワークのSpringBootも使用してみたいです。 セキュリティ面の対策が甘い状態なのでこれを機にしっかり知識を固めなければと思いました。

タコゲーム

GitHub

「タコゲーム」

https://github.com/kanisan09/portfolio-Javascript.git

説明

職業訓練校の基本情報技術者養成科ではJavaScriptは触れないのですが、なにか作ってみたいの気持ちで作りました。 画面上の動きを表現できる所に着目し日本出身のタコさんを探すゲームを作りました。 世界のタコ漁獲量統計サイトで調べた上位国のタコ(中国、メキシコなど)を参加させました。 同じ位置のタコを当たり指定してしまうと面白みに欠けるかもと考えまして、タコの結果自体をランダムにしてどのタコが当たりという概念をなくしました。タコ単体での動きはタコ自体をgif画像にして動かしています。

制作時間

3時間
