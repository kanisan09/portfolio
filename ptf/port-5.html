Java portfolio

説明

サーブレットとjspを使用してつぶやき機能がある動的アプリケーションを作りました。

使用したもの

統合開発環境 Eclipse / tomcat

使用サーバー

AWS EC2

インスタンスやポートの開放設定、Amazon inux 2023にmysqlやtomcatをインストール、起動確認など無事サーバーに上げることはできましたが毎日不安との戦いでした。
無駄な事もたくさんしましたしかなり時間をかけてしまいましたがこの経験は一生忘れないと思います。

制作時間

1ヵ月半

反省点、改善点

動的アプリケーションがどのように作られているのか理解はできたが、セッションがうまくきいておらず投稿文がスコープの中に残ったままになってしまった。
H2データベースを練習でつないでみたりしましたが、MYSQLなどのDBをつないで、ログインや投稿内容がうまく記憶できるようにしたいです。
DAOやxmlファイルの理解も深めたい。フレームワークのSpringBootも使用してみたい。 セキュリティ面の対策が甘い状態なのでこれを機にしっかり知識を固めなければと思いました。
